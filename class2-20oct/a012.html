Median

Task 1

Write a Python method median(a: int, b: int, c: int) -> int, that calculates the median of the three given numbers (it has to be one of those numbers!).
Example: median(25, 11, 33) == 25 and median(1, 1, 2) == 1

Task 2

Now do the same again, but think of an alternative to your first solution. Write a Python method median2(a: int, b: int, c: int) -> int.

Task 3

How would you test if your functions work properly? Give some tests within your script. These will be discussed during the exercise.
