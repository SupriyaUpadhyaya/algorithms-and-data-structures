Transformation of Decimal and Dual Numbers

  1. Write a Python method transform_to_dual(dec: int) -> str that transforms a number $n > 0$ given in base10 to the String representation of that same number in base2. Be aware of the order of bits!

  2. Test your method appropriately.

Example: transform_to_dual(13) == "1101".

Note:

  • To solve this task do not use any library function such as "{0:b}".format(x). You may however use that function to validate your results.

  • To create the output String you can concatenate two Strings by using the + operator.
